Overview:
This is a Python GUI application for currency conversion using the Tkinter library. It fetches currency exchange rates from the Open Exchange Rates API and allows users to convert one currency to another. The application also allows users to reset the input fields and displays a greeting message upon exit.

Installation:

Clone the repository to your local machine.
Ensure that you have Python 3.x installed on your machine.
Install the required dependencies using pip by running the command "pip install requests".
Run the application using the command "python currency_converter.py".

Here are the steps to use the converter:

Enter the amount you want to convert in the "Enter Amount" field.
Select the currency you want to convert from in the "From Currency" dropdown.
Select the currency you want to convert to in the "To Currency" dropdown.
Click the "Convert" button to perform the conversion.
The converted amount will be displayed in the output field.
To reset the input fields, click the "Reset" button.
Upon exit, a greeting message will be displayed.
